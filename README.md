# Simon says game

This is a fun and challenging Simons Says game project! It's a classic memory game where players repeat a sequence of lights displayed by the game. Test your memory and reflexes.

Useing the Project:

.Playing the game: Press any key and start the game.
.Learn from the code: The code provides a good example of implementing core game mechanics like sequence generation and scoring.
.Contribute and Extend: Feel free to modify the code to add features or improve the game!

functionality:

.Generates random sequences of light.
.Plays the sequences for the player to remember.
.provides feedback on whether the player's input is correct.
.tracks the layer's score and increases.
